# Sample Project: TaskFlow CLI Tool

## Overview
TaskFlow is a lightweight command-line task manager designed to help users create, track, and complete daily tasks efficiently.

## Features
- Create tasks
- Mark tasks as completed
- Delete tasks
- List all tasks
- Lightweight and fast

## Installation

```bash
git clone https://github.com/example/taskflow.git
cd taskflow
npm install
